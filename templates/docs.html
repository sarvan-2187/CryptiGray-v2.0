üîê Text Encryption

To encrypt a text message using CryptiGray:

1

Navigate to the Encryption Page

Click on the "Encrypt" link in the navigation menu or go to /encrypt-page.

2

Enter Your Message

Type or paste the text you want to encrypt in the "Your Message" textarea.

Example: Hello World!
3

Provide a Secret Key

Enter a strong secret key that will be used for encryption. Remember this key as you'll need it for decryption.

Example: mySecretKey123
4

Encrypt Your Message

Click the "Encrypt Message" button. Your encrypted text will appear in the output area.

5

Copy or Download

Use the "Copy Encrypted Text" or "Download as File" buttons to save your encrypted message.

üí° Important Note

The encryption process follows these steps:

  1. Convert each character to its ASCII value
  2. Convert ASCII to 8-bit binary
  3. Transform binary to Gray code
  4. Apply XOR operation with your secret key

Always use a strong key and keep it secure for decryption.

üîì Text Decryption

To decrypt an encrypted message:

1

Navigate to the Decryption Page

Click on the "Decrypt" link in the navigation menu or go to /decrypt-page.

2

Paste Encrypted Text

Paste the encrypted binary sequence into the "Encrypted Text" textarea.

Example: 10010110 11001010 01101001 10010110 01101001
3

Enter the Secret Key

Provide the exact same secret key that was used for encryption.

4

Decrypt Your Message

Click the "Decrypt Message" button. Your original text will appear if the key is correct.

‚ö†Ô∏è Important

Decryption will only work with the exact same key used for encryption. The process is case-sensitive and requires the key to match precisely.

üìÑ PDF Encryption

CryptiGray also allows you to encrypt PDF files with a password:

1

Go to the PDF Encryption Page

Click on the "PDF" link in the navigation menu or go to /pdf-page.

2

Select a PDF File

Click the upload area to select a PDF file or drag and drop a PDF onto the area.

3

Set a Password

Enter a strong password that will be used to encrypt the PDF.

4

Encrypt the PDF

Click the "Encrypt PDF" button. The encrypted PDF will be available for download once processing is complete.

üîí Security Note

The PDF encryption uses standard PDF encryption methods. Remember your password as there is no way to recover it if lost.

üîß Troubleshooting

Common Issues

Decryption Not Working

If decryption fails, ensure:

  • You're using the exact same key used for encryption
  • The key is case-sensitive (e.g., "Key" ‚â† "key")
  • The encrypted text was copied completely and correctly

PDF Encryption Issues

If PDF encryption fails:

  • Ensure you're uploading a valid PDF file
  • Check that the file is not corrupted
  • Try with a smaller PDF if encountering performance issues

Performance Problems

For very long texts or large PDFs:

  • Encryption/decryption may take longer
  • Consider breaking very long texts into smaller parts
  • For PDFs, try files under 50MB for best performance
